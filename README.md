# PowerBiTrabaoPracticoII
Segundo Laboratorio DATA ANALYTICS 2025
En la etapa de Transformación (ETL), en primer lugar corregí los tipos de datos correspondientes a las columnas de país y ciudad, asegurando su categorización geográfica para que puedan ser utilizadas correctamente en el visual de mapa del informe. Además, ajusté los tipos de datos numéricos y de fecha para garantizar cálculos consistentes y evitar errores en el modelado.

Luego, creé una tabla de medidas con el objetivo de centralizar los cálculos principales y mantener el modelo más ordenado y fácil de mantener. También generé una tabla de calendario ("Calendar"), marcada como tabla de fecha, que permite trabajar con funciones de Time Intelligence y realizar análisis temporales más precisos, como comparaciones entre períodos, acumulados y tendencias.

Por último, desarrollé una serie de medidas DAX esenciales para el análisis, entre ellas: promedio neto por transacción (AVG Net per Transaction), cantidad total vendida (Total Quantity Sold), ventas totales (Total Sales), total de transacciones (Total Transactions) y clientes únicos (Unique Customers). Estas medidas sirven como base para construir los KPI y visualizaciones del dashboard final.
